For the image above, I used the :hover pseudo-class to make the image react in a dramatic way when the user hovers over it. I employed the properties transform and box-shadow to achieve this effect. Transform is a property that allows you to scale, rotate, or translate an element. Box-shadow is a property that allows you to add a shadow to an element. By employing both of these, I created a 3D effect that makes the image turn and have a glowing shadow when the user hovers over it.

Video Recommendations

The internet is home to a panoply of content, but there are some real diamonds in the rough if you know where to look. Here are some S-tier YouTube video recommendations that will make you laugh, cry, and ponder.
To create these embedded YouTube videos, I employed the help of the iframe tag. This HTML tag is a cool way to embed HTML content from other websites into your own page. The i stands for inline frame which means that it puts content from another website into that space on your webpage. It has a plethora of attributes that can be used to customize the embedded content and what functionality the content has within the context of the parent page. Here I had it embed the YouTube videos in the page and allowed them to have some classical functionality like autoplay, picture-in-picture, and fullscreen.
I also used the CSS counter-reset and counter-increment properties with the :before pseudo-element to number the video titles. This was a useful trick to number the videos without having to make the headers a numbered list, which would have made formatting the page to look the way I wanted significantly more difficult.
